Wymadany jest nodejs (v20.18.0 lub wyżej) i npm
Aby otworzyć projekt należy użyć najstępujących komend (bedac w edytorze lub na scieszce pliku File Manager w cmd):

npm install
npm start

Uwagi:
    Aplikacja długo sie ładuje za pierwszym razem!
    Aplikacja nie modyfikuje żadnych plików na dysku!
